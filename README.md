# simple-calculator

Dies ist ein simpler Taschenrechner der folgende Rechenoperationen durchführen kann:
-Addition
-Subtraktion
-Multiplikation
-Division

Das Projekt dient dazu, Phyton Grundlagen zu lernen. 
Es beinhaltet folgende Punte:
- If/else
- Funktionen
- try/except Fehlermeldungen
- input
